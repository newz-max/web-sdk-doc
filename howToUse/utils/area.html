# style

选项类型描述
strokeBoolean 是否沿路径绘制边框。把它设置为false ，可以禁用多边形或圆形的边框。
colorStringStroke（描边）颜色
weightNumberStroke（描边）宽度，单位：像素
opacityNumberStroke（描边）的不透明度
lineCapString 定义描边 结束时使用的形状open in new window 的字符串。
lineJoinString 定义在描边 转角处使用的形状open in new window 的字符串。
dashArrayString 一个定义描边的点画线的 图案范式open in new window的字符串。 在一些旧的浏览器open in new window中，对 Canvasopen in new window-powered 图层不起作用。
dashOffsetString 一个字符串，它定义了进入 dash 范式的距离，以 dash 开始open in new window。 在一些旧的浏览器open in new window中，对 Canvasopen in new window-powered 图层不起作用。
fillBoolean 是否对路径进行颜色填充。设置为 false 可以禁止对多边形或圆形的填充。
fillColorString填充颜色
fillOpacityNumber填充的不透明度。
fillRuleString 一个字符串，定义 形状的内部区域open in new window 是如何被确定的。
classNameString在一个元素上设置的自定义类名。仅适用于 SVG 渲染器。
